## What I learned
- A JSON endpoint follows the same Symfony flow as an HTML page: route -> controller action -> response, but the response format is JSON.
- `$this->json()` is the controller shortcut for returning JSON.
- A model-style class is just a plain PHP class used to structure data; it is not automatically an entity.
- A service is an object whose main job is to do work for the application.
- Autowiring means Symfony injects a matching service based on the type-hint.
- An interface defines the contract, while the injected object may be a concrete implementation such as `Monolog\Logger`.
- Constructor injection is the standard way to inject services into other services.
- Twig can work with both arrays and objects, using getters behind the scenes for object access.
