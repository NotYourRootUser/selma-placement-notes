# selma-php-prep

Prep notes and small PHP practice files for ramping into PHP, LAMP, and Symfony before starting my Selma placement.

## Purpose
This repository tracks my prep work before starting placement with Selma. The goal is to build enough PHP, LAMP, and Symfony familiarity to enter the placement with better technical context and a cleaner mental model of the stack.

- LAMP stack basics

- small practice files and local testing
